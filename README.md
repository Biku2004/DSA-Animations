## Questions

- **Prefix Sum Tradeoff**: Understand the memory-time tradeoff in prefix sum. See how preprocessing takes O(N) time but enables O(1) queries.
- **Sliding Window**: Visualize the sliding window technique for finding maximum length subarrays with a target sum.

## Features

- **Dark Mode**: Toggle between light and dark themes (Context API powered).
- **Responsive Design**: Works seamlessly on desktop and mobile.

## Tech Stack

- React 19
- Tailwind CSS 4
- Vite
- Lucide Icons
